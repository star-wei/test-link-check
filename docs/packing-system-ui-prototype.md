## 6. 关键交互说明

### 6.1 异常标记规则
- 🔴 红色：尺寸超限，必须处理
- 🟡 黄色：建议优化，可忽略
- 🟢 绿色：正常，无需干预

### 6.2 拖拽操作
- 产品列表 → 箱子区域：添加到该箱
- 箱子A → 箱子B：移动产品
- 箱内产品：调整摆放顺序

### 6.3 快捷键
- `Ctrl + R`：重新计算
- `Ctrl + S`：保存草稿
- `Ctrl + E`：导出清单
- `Delete`：删除选中产品

### 6.4 实时反馈
- 调整尺寸后自动重新计算
- 超重时红色警告并提示分箱
- 利用率低于50%时建议优化
