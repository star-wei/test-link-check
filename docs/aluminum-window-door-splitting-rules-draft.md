# 铝合金门窗拆件规则草案

> 基于通用行业知识整理，供用户（行业专家）纠正和补充。

---

## 一、固定窗（Fixed Window）

### 可拆性
**可拆**，且是发货前最常见的拆件类型。

### 拆分逻辑
- 大尺寸的固定窗通常按 **宽度方向** 等分成若干 "固定扇"（或称 "固定玻璃单元"）
- 常见拆分宽度：1000mm ~ 1200mm 一扇（具体取决于系列和玻璃厚度）
- 边扇可能因包含框料而比中间扇宽 20~50mm
- 高度方向通常 **不拆**（保持整体高度）

### 关键部件
- 玻璃和框料通常 **已组装完成** 后发货（玻璃已扣入扇框）
- 部分项目为降低破损风险，可能将玻璃单独包装，但多数情况下固定窗是 **整扇发货**
- 压条、扣盖、密封胶条等辅材通常单独包装成一箱配件

---

## 二、推拉门 / 移门（Sliding Door / Patio Door）

### 可拆性
**部分可拆**，取决于尺寸和轨道系统设计。

### 拆分逻辑
- **活动扇（门扇）**：每扇宽度通常 ≤ 1500mm，可直接作为独立组件
- **固定扇（非活动部分）**：若宽度较大，可能拆为 2~3 扇固定玻璃单元
- **轨道（上轨 + 下轨）**：
  - 通常 **不拆短**，保持出厂长度
  - 若轨道长度 > 集装箱对角线（约 5.8m）或货车限长，则必须分段
  - 上轨/下轨一般独立包装，避免扇料压坏轨道
- **边框（光企/勾企）**：通常与门扇组装在一起发货

### 滑轨相关问题
- 推拉门的滑轮、锁具、防撞块等五金件通常 **单独成箱**
- 轨道槽内不能受压，否则影响推拉顺畅度
- 毛条、滑轮、锁扣等易损件需要防震包装

---

## 三、平开门（Swing Door / Casement Door）

### 可拆性
**小尺寸通常不拆**，大尺寸固定侧可能拆分。

### 拆分逻辑
- 门扇本身作为一个组件（含玻璃）
- 若带固定侧窗（如门连窗），固定侧窗按固定窗逻辑拆分
- 门框（上框、下框、边框）通常 **成套发货**，不拆散
- 合页、门锁、拉手等五金件可单独成箱

---

## 四、折叠门（Folding Door / Bifold Door）

### 可拆性
**按折叠扇数拆分**，每扇为独立组件。

### 拆分逻辑
- 折叠门由多扇活动门扇通过铰链连接，运行时沿轨道滑动折叠
- 发货时，**每扇折叠门扇作为独立组件**
- 轨道系统（上轨 + 下轨）通常整根发货，过长时分段
- 滑轮组、铰链、拉手、锁具等 **单独成箱**

### 滑轨相关
- 折叠门对轨道的平整度要求极高，轨道通常用木框保护
- 多轨道系统（如三轨折叠门）轨道数量多，可能需要分 2~3 箱运输

---

## 五、入户门（Entry Door）

### 可拆性
**基本不拆**，整樘门作为一个组件发货。

### 原因
- 入户门结构复杂，含门框、门扇、锁具、猫眼、门铃等
- 玻璃（若有）通常已预装在门扇内
- 拆分会导致现场二次组装困难，且破坏密封和防盗性能
- 包装方式：整扇门 + 木框架保护

---

## 六、哪些部件通常单独包装？

| 部件 | 是否单独包装 | 原因 |
|---|---|---|
| **玻璃（未安装）** | 是 | 易碎，需木箱/护角保护 |
| **滑轮/轨道** | 是 | 轨道不能受压，滑轮易损 |
| **五金件（锁、把手、合页）** | 是 | 小件集中成箱，避免丢失 |
| **纱网/纱窗扇** | 是 | 通常单独一箱，防止挤压变形 |
| **密封胶条/压条** | 是 | 长条形，单独成卷或成箱 |
| **门扇/窗扇（已组装玻璃）** | 否 | 作为主体组件直接装箱 |

---

## 七、拆件时的关键尺寸约束

### 1. 玻璃尺寸与框料配合
- 拆分后单扇的宽度需预留框料搭接量（通常每边 20~40mm）
- 玻璃尺寸 = 扇宽 - 左右框料厚度 - 安装间隙

### 2. 轨道长度限制
- 标准 20GP 集装箱内长：约 5.89m
- 40GP/40HQ 集装箱内长：约 12.03m
- 货车车厢长度：约 9.6m / 13.5m / 17.5m
- 铝型材轨道出厂最长可达 6m，超过则需 **现场拼接** 或 **分段发货**

### 3. 单件重量限制
- 人工搬运：通常 ≤ 80kg/人
- 叉车搬运：单件 ≤ 1500kg（本次系统约束）
- 玻璃组件密度高，大尺寸固定窗单扇可达 200~400kg

3. **轨道是否分段**：什么情况下轨道必须截断发货？
4. **折叠门的扇宽标准**：每扇折叠门通常多宽？
5. **铝木/E80 等定制产品**：是否完全不拆，还是按特定规则拆分？
6. **玻璃与扇框的关系**：发货时玻璃是否已经扣入扇框？还是玻璃单独发？
7. **辅材包装**：五金、轨道、胶条等是每单独立成箱，还是按工厂统一打包？
